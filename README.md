# Sharma Enterprises - Premium Healthcare Imaging Solutions Website

A stunning and premium website for Sharma Enterprises, a leading provider of medical printer installation, diagnostic imaging film supply, technical support, and imaging equipment solutions for hospitals and diagnostic centers.

## Overview

This website is built using the technology stack recommended in the project documents:
- **Frontend**: Next.js 14 (React) with Tailwind CSS and shadcn/ui
- **Animations**: Framer Motion
- **Styling**: Custom medical-themed design with deep blue color scheme
- **Features**: Responsive design, service showcase, contact forms, trust sections, portfolio display

## Getting Started

Note: This codebase is designed to demonstrate the website structure and design. To run it locally:

1. Install Node.js (version 18 or higher recommended)
2. Clone or copy this repository
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Visit `http://localhost:3000` to view the website
